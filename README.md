# KartInfo

* 🔑 GraphQL로 래핑한 카트라이더 오픈API를 활용하여 유저 정보 조회 웹페이지 제작
* 🔗 웹페이지 링크 : https://kartinfoweb.herokuapp.com/

### ⭐ 사용한 API
* 카트라이더 오픈 API : https://developers.nexon.com/kart

### 🚀 기술스택
* React, GraphQL, Apollo, webpack 사용
* heroku deploy
* 자세한 구현법은 추후 포스팅할 예정입니다
